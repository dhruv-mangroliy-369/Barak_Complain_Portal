
# ComplainEase


ComplainEase is a complaint portal developed by Dhruv Mangroliya for the residents of Barak Hostel, IIT Guwahati. This platform allows residents to easily lodge and track complaints related to hostel maintenance and services.

🚀 Getting Started
To run this project on your local server, follow these steps:

1. Clone the Repository
First, clone the project repository to your local machine using the following command:

-> git clone https://github.com/dhruv-mangroliya/ComplainEase.git

2. Navigate to the Project Directory
Change your working directory to the project folder:

-> cd ComplainEase

3. Install Dependencies
Install the necessary dependencies by running:

-> npm install

4. Start the Development Server
Start the development server using:

-> npm start

Your application should now be up and running locally.

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript, React.js

Backend & Authentication: Firebase

🎯 Purpose

ComplainEase is designed to streamline the complaint management process within Barak Hostel. It offers an easy-to-use interface for students to report issues, and it provides administrators with tools to manage and address these complaints effectively.
